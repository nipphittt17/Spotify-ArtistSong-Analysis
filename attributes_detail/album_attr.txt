1. album_type
string
Required
The type of the album.
Allowed values: "album", "single", "compilation"
Example: "compilation"

2. total_tracks
integer
Required
The number of tracks in the album.
Example: 9

3. available_markets
array of strings
Required
The markets in which the album is available: ISO 3166-1 alpha-2 country codes. NOTE: an album is considered available in a market when at least 1 of its tracks is available in that market.
Example: ["CA","BR","IT"]

4. external_urls
object
Required
Known external URLs for this album.

5. href
string
Required
A link to the Web API endpoint providing full details of the album.

6. id
string
Required
The Spotify ID for the album.
Example: "2up3OPMp9Tb4dAKM2erWXQ"

7. images
array of ImageObject
Required
The cover art for the album in various sizes, widest first.

8. name
string
Required
The name of the album. In case of an album takedown, the value may be an empty string.

9. release_date
string
Required
The date the album was first released.
Example: "1981-12"

10. release_date_precision
string
Required
The precision with which release_date value is known.
Allowed values: "year", "month", "day"
Example: "year"

11 restrictions
object
Included in the response when a content restriction is applied.

12. type
string
Required
The object type.
Allowed values: "album"

13. uri
string
Required
The Spotify URI for the album.
Example: "spotify:album:2up3OPMp9Tb4dAKM2erWXQ"

14. artists
array of SimplifiedArtistObject
Required
The artists of the album. Each artist object includes a link in href to more detailed information about the artist.
